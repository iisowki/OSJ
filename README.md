# JasonTheWeen Digital Fanzine

A prototype of a digital fanzine showcasing JasonTheWeen's music, albums, and history.

## Features

- Album discography with interactive carousel
- Popular songs section
- History timeline
- About section with social media links

## Files

- `OSJProject.html` - Main HTML file
- `OSJStyling.css` - Stylesheet
- `albumSwitch.js` - Album carousel functionality
- `assets/` - Images, fonts, logos, and videos

## Setup

Simply open `OSJProject.html` in a web browser.
